1. Искусственный интеллект — не магия

Искусственный интеллект — это область технологий, где компьютерные системы выполняют задачи, похожие на интеллектуальные: понимают текст, распознают речь, находят закономерности, делают прогнозы или создают новый текст.

Но AI не обязательно “думает” как человек. Чаще всего он обрабатывает данные и выдает результат на основе правил, обучения и вероятностей.

2. Обычная программа и AI-система

Обычная программа работает по заранее прописанным правилам: если произошло A, выполнить B. AI-система анализирует входные данные и выбирает наиболее вероятный результат.

Обычная программа:
если пароль неверный → показать ошибку

AI-система:
входные данные → анализ → вероятный результат

3. Что такое модель

Модель — это система, обученная на данных для выполнения определенного типа задач. Она получает входные данные и выдает результат.

Вход → Модель → Выход

Текст вопроса → Языковая модель → Текст ответа
Фото → Модель распознавания → Название объекта
Данные о погоде → Прогнозная модель → Прогноз

Для промпт-инженера важно: модель получает не ваше скрытое намерение, а конкретный текст.

Практика 3. Что является входом для LLM?

4. Что такое LLM

LLM означает Large Language Model — большая языковая модель. Это AI-модель, обученная работать с языком. Она принимает текст на вход и генерирует текст на выходе.

LLM может объяснять темы, писать письма, составлять планы, переписывать тексты, анализировать документы, генерировать идеи и помогать с кодом.

5. Почему LLM не обычный чат-бот

Старый сценарный чат-бот часто выбирает готовую фразу из заранее написанного дерева ответов. LLM строит ответ заново, учитывая запрос, контекст, стиль, формат и ограничения.

Сценарный чат-бот
  • работает по готовым сценариям;
  • плохо реагирует на нестандартные запросы;
  • обычно не умеет гибко менять стиль ответа.
LLM
  • генерирует ответ заново;
  • учитывает контекст;
  • может менять формат, стиль и уровень сложности;
  • может ошибаться, если задача поставлена плохо.

Практика 5. Выберите правильное утверждение

6. Что такое промпт

Промпт — это инструкция, которую пользователь передает модели. Хороший промпт — это не просто вопрос, а точная постановка задачи.

Ты — [роль].
Сделай [задача].
Аудитория: [для кого].
Формат ответа: [структура].
Ограничения: [чего избегать].

Практика 6. Найдите, чего не хватает

Дан слабый промпт:

7. Какой промпт лучше

Сильный промпт снижает неопределенность. Он говорит модели, кем быть, что сделать, для кого, в каком формате и с какими ограничениями.
